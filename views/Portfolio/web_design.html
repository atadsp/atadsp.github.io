Web Design and Coding Experience

MEAM.JS Boiler Plate - Super Duper Mart

There is a fantastic template out there that allows web developers to quickly scrub out websites using the MEAN stack. Theses sites are basic and require a lot of additional work to get to any sort of professional standard. I took this boiler plate and added additional functionality to it. Primarily it had to do with interlinking the various sub-pages so that the various tables on the database could talk with one another. The theme of the website is based of the video game franchise called Fallout, produced by Interplay Entertainment, Black Isle Studio, Micro Forte, Bethesda Game Studios, and Obsidian Entertainment.
